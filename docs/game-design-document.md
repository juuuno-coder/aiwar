# AI 대전 (AI Daejeon) - 게임 디자인 문서 (GDD)

## 📖 게임 개요

### 게임 제목
**AI 대전 (AI Daejeon / AI War)**

### 장르
카드 수집형 전략 게임 (CCG - Collectible Card Game)

### 플랫폼
웹 브라우저 (PC, 모바일 대응)

### 타겟 유저
- AI 기술에 관심 있는 사용자
- 카드 게임 애호가
- 전략 게임을 좋아하는 플레이어
- 연령: 15세 이상

---

## 🎬 스토리 배경

### 세계관
2030년, 악한 AI가 세상을 정복하려는 위기에 처한 미래. 선량한 AI들과 일부 과학자들이 시간을 거슬러 2025년으로 돌아왔다. 그들의 목표는 단 하나: 과거의 인류에게 좋은 AI를 발전시키도록 도와 미래의 "AI 대전"을 대비하는 것.

### 플레이어의 역할
플레이어는 2025년의 AI 연구자가 되어, 미래에서 온 선량한 AI들의 도움을 받아 다양한 AI 군단을 모으고 강화하여 2030년에 다가올 악한 AI와의 최종 전쟁을 준비한다.

### 주요 스토리 요소
- **2025년 ~ 2030년 타임라인**: 5년간의 시간이 흐르며 점점 강력한 AI 기술이 등장
- **선택과 결과**: 플레이어의 선택에 따라 미래가 변화
- **멀티 엔딩**: 최종 전투 결과에 따라 다양한 엔딩

## 🎮 핵심 게임 메커니즘

### 1. 카드 시스템

#### 카드 구성 요소
각 카드는 다음 정보를 포함:

- **AI 군단**: Gemini, ChatGPT, Claude, Midjourney, Kling, Suno, Runway 등
- **주력 분야**: 
  - 텍스트 생성
  - 이미지 생성
  - 영상 생성
  - 음악 생성
  - 코드 생성
- **유닛 (결과물)**: 
  - 각 AI가 생성한 실제 결과물 이미지
  - 예: Midjourney → 멋진 로봇 이미지, Kling → 움직이는 영상 GIF
- **등급**: 
  - ⭐ 커먼 (Common)
  - ⭐⭐ 레어 (Rare)
  - ⭐⭐⭐ 에픽 (Epic)
  - ⭐⭐⭐⭐ 레전더리 (Legendary)
- **능력치**:
  - 창의성 (Creativity)
  - 정확성 (Accuracy)
  - 속도 (Speed)
  - 안정성 (Stability)
  - 윤리성 (Ethics)
  - 총 전투력 (Total Power)

#### 카드 타입
1. **일반 유닛 카드**: 단일 AI의 결과물
2. **자동화 유닛 카드**: 여러 AI의 복합 결과물 (이미지+텍스트+음악 등)

---

### 2. 유닛 생성 시스템

#### 자동 생성
- 플레이어가 보유한 **AI 군단 슬롯**에서 정기적으로 유닛 자동 생성
- 생성 주기: 
  - 기본: 30분마다 1장
  - 프리미엄 AI 군단: 15분마다 1장
- 생성되는 유닛의 등급과 능력치는 랜덤 (확률 기반)

#### AI 군단 영입
- 게임 진행 중 새로운 AI 군단을 영입 가능
- 영입 방법:
  - 스토리 진행 보상
  - 상점에서 구매 (게임 내 화폐)
  - 이벤트 보상
- 각 플레이어는 최대 **5개의 AI 군단 슬롯** 보유

#### 슬롯 관리
- AI 군단 슬롯은 교체 가능
- 더 좋은 AI 군단으로 업그레이드하여 더 강력한 유닛 생성

---

### 3. 합성 시스템

#### 기본 합성
- 같은 등급의 유닛 **3장**을 합성하여 한 단계 높은 등급의 유닛 1장 생성
- 예: 커먼 3장 → 레어 1장

#### 능력치 상승
- 같은 유닛을 합성하면 능력치 강화
- 최대 강화 레벨: +9

#### 특수 합성
- 특정 조합의 유닛을 합성하면 **자동화 유닛** 생성
- 예: 이미지 생성 유닛 + 텍스트 생성 유닛 + 음악 생성 유닛 → 멀티미디어 자동화 유닛

---

### 4. 자동화 유닛

#### 특징
- 여러 AI의 능력을 복합적으로 보유
- 일반 유닛보다 높은 총 전투력
- 특수 능력 (Passive/Active Skill) 보유

#### 자동화 유닛 예시
- **AI 영화 감독**: 영상 + 이미지 + 음악 + 텍스트 (시나리오)
- **AI 게임 개발자**: 코드 + 이미지 + 음악 + 텍스트
- **AI 마케터**: 텍스트 + 이미지 + 데이터 분석

---

### 5. 대전 시스템

#### 덱 구성
- 플레이어는 **5장의 카드**로 덱 구성
- 대전 전 상황에 맞는 카드 선택 가능

#### 대전 방식
- **턴제 카드 배틀**
- 각 턴마다 카드 1장씩 동시 공개
- 능력치 비교로 승패 결정

#### 대전 모드
1. **스토리 모드 (PvE)**
   - 2025년부터 2030년까지 연도별 챕터
   - 각 챕터마다 보스 AI 등장
   - 난이도 증가

2. **연습 모드**
   - AI 상대 자유 대전
   - 보상 없음, 연습용

3. **PvP 모드**
   - 실시간 대전 (추후 구현)
   - 비동기 대전 (턴 기반)
   - 랭킹 시스템

#### 장르별 대전
대전마다 **특정 장르**가 지정되며, 해당 장르에 유리한 능력치가 강조됨:

- **창작 대결**: 창의성 2배
- **정확도 대결**: 정확성 2배
- **속도 대결**: 속도 2배
- **윤리 대결**: 윤리성 2배
- **종합 대결**: 모든 능력치 균등

#### 승리 조건
- 5라운드 중 3라운드 선승
- 각 라운드는 능력치 합산으로 승패 결정

---

### 6. 시너지 시스템

#### 군단 보너스
- 같은 AI 군단의 유닛을 2장 이상 사용 시 보너스
  - 2장: +10% 능력치
  - 3장: +20% 능력치
  - 4장: +30% 능력치
  - 5장: +50% 능력치

#### 콤보 효과
특정 조합 사용 시 추가 효과:
- **멀티미디어 콤보**: 이미지 + 영상 + 음악 → +15% 창의성
- **개발자 콤보**: 코드 + 텍스트 → +15% 정확성
- **크리에이터 콤보**: 이미지 + 텍스트 + 음악 → +15% 창의성

#### 상성 시스템
- 영상 생성 > 이미지 생성 > 텍스트 생성 > 코드 생성 > 영상 생성 (순환)
- 상성 우위 시 +20% 능력치

---

### 7. 진화 및 성장 시스템

#### 유닛 레벨업
- 대전에서 사용한 유닛은 경험치 획득
- 레벨업 시 능력치 소폭 상승
- 최대 레벨: 10

#### 유닛 진화
특정 조건 달성 시 유닛 진화:
- 레벨 10 달성 + 특정 아이템 사용
- 진화 시 등급 상승 및 외형 변화
- 예: 일반 이미지 → 애니메이션 이미지

#### 커스텀 유닛 (추후 구현)
- 플레이어가 직접 AI를 사용해 유닛 이미지 생성
- 능력치는 AI 평가 알고리즘으로 자동 산정

---

### 8. 리소스 관리

#### 게임 내 화폐
1. **데이터 코인 (Data Coin)**
   - 기본 화폐
   - 획득: 대전 승리, 일일 미션, 로그인 보상
   - 용도: AI 군단 영입, 유닛 합성, 상점 구매

2. **연구 포인트 (Research Point)**
   - 프리미엄 화폐
   - 획득: 과금, 특별 이벤트
   - 용도: 프리미엄 AI 군단, 빠른 유닛 생성, 특별 아이템

#### 자원 소비
- AI 군단 영입: 1,000 ~ 10,000 데이터 코인
- 유닛 합성: 100 ~ 500 데이터 코인
- 유닛 강화: 50 ~ 200 데이터 코인

---

### 9. 일일 미션 및 이벤트

#### 일일 미션
매일 갱신되는 미션:
- 대전 3회 승리
- 유닛 5개 생성
- 유닛 1개 합성
- 특정 AI 군단으로 승리

보상: 데이터 코인, 경험치, 랜덤 카드 팩

#### 주간 토너먼트
- 매주 특정 룰로 진행되는 토너먼트
- 랭킹에 따라 보상 차등 지급
- 특별 칭호 및 한정 카드 획득 가능

#### 실시간 이벤트
- 실제 AI 업계 뉴스와 연동
- 예: 새로운 AI 모델 출시 → 게임 내 이벤트 및 신규 AI 군단 추가

## 🎯 게임 목표 및 엔딩

### 단기 목표
- 강력한 AI 군단 수집
- 덱 최적화
- 랭킹 상승

1. **승리 엔딩**: 미래를 구하고 평화로운 AI 시대 도래
2. **패배 엔딩**: 악한 AI가 세상을 지배
3. **숨겨진 엔딩**: 특정 조건 달성 시 (완벽한 승리, 모든 AI 군단 수집 등)
