# AI 대전 - UI/UX 플로우

## 🎨 개요
이 문서는 AI 대전 게임의 화면 구성과 사용자 흐름을 정의합니다.

---

## 📱 화면 구성

### 1. 시작 화면 (Splash Screen)

**요소**:
- 게임 로고 (애니메이션)
- "AI 대전" 타이틀
- 로딩 바
- 버전 정보

**전환**:
- 로딩 완료 → 로그인 화면 (미로그인 시) 또는 메인 화면 (로그인 시)

---

### 2. 로그인/회원가입 화면

#### 2.1 로그인
**요소**:
- 이메일 입력 필드
- 비밀번호 입력 필드
- "로그인" 버튼
- "회원가입" 링크
- "비밀번호 찾기" 링크
- 소셜 로그인 버튼 (선택)

#### 2.2 회원가입
**요소**:
- 이메일 입력 필드
- 비밀번호 입력 필드
- 비밀번호 확인 필드
- 닉네임 입력 필드
- "가입하기" 버튼
- "이미 계정이 있으신가요?" 링크

**전환**:
- 로그인 성공 → 메인 화면
- 회원가입 성공 → 튜토리얼 (첫 플레이) 또는 메인 화면

---

## 🔄 사용자 플로우

### 첫 플레이 플로우
```
시작 화면
  ↓
회원가입
  ↓
튜토리얼 (스토리 소개)
  ↓
AI 군단 슬롯 설명
  ↓
첫 유닛 생성 (무료)
  ↓
덱 구성 가이드
  ↓
첫 대전 (튜토리얼 보스)
  ↓
합성 시스템 설명
  ↓
메인 화면
```

### 일반 플레이 플로우
```
메인 화면
  ↓
AI 군단 관리 → 유닛 수령 → 인벤토리
  ↓
인벤토리 → 합성/강화
  ↓
덱 구성
  ↓
스토리 모드 또는 대전
  ↓
대전 결과 → 보상 획득
  ↓
메인 화면
```

### 일일 미션 플로우
```
메인 화면 (미션 알림)
  ↓
일일 미션 화면
  ↓
미션 확인
  ↓
게임 플레이 (대전, 합성 등)
  ↓
미션 완료
  ↓
보상 수령
```
